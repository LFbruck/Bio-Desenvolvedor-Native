# TP3 - Desenvolvimento Mobile com React Native
**Aluno:** Luiz Filipe Bruck Reis Mattos

## 🚀 O Projeto
Aplicativo de perfil profissional desenvolvido com Expo e React Native, demonstrando o uso de navegação complexa e consumo de APIs externas.

## 🛠️ Funcionalidades
- **Arquitetura Organizada:** Divisão clara entre telas e navegação na pasta `src`.
- **Navegação Stack:** Fluxo de telas entre Home e Perfil Detalhado.
- **Navegação Bottom Tabs:** Menu inferior para acesso a Projetos e Contato.
- **Integração com API:** Listagem dinâmica de repositórios reais via API do GitHub.
- **Estilização:** Layout responsivo utilizando StyleSheet e Flexbox.

## 📦 Como rodar
1. Clone o repositório.
2. Execute `npm install` para instalar as dependências.
3. Execute `npx expo start` para iniciar o projeto.
